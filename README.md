## Lo que estoy construyendo

### Crypto Trading Scanner
Sistema automatizado de señales para los 20 pares de cripto con mayor capitalización. Análisis multi-timeframe (4H → 1H → 5M), puntuación de señales y alertas en tiempo real por Telegram.

- **Backend:** Python 3.12 · FastAPI · SQLite
- **Frontend:** React 18 · TypeScript · Vite · lightweight-charts
- **Infraestructura:** Docker · Windows Task Scheduler
- **Fuente de datos:** Binance Futures (fallback: Bybit)
